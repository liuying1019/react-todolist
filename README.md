# react-todolist
todolist
用react完成一个纯前端应用TodoList，实现以下需求：
渲染所有代办事项；
可以添加新的代办事项；
可以将已经完成的事项标记为已完成；
区分已完成和未完成事项；
可以删除事项。
使用了antd react
